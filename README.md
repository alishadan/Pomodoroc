\# Pomodoro Timer with Audio



A command-line Pomodoro timer application with audio notifications using miniaudio.



\## Features

\- Pomodoro timer with work/break intervals

\- Audio notifications

\- Configurable settings

\- Cross-platform support



\## Building



\### Using CMake (Recommended):

```bash

\# Clone repository

git clone <your-repo-url>

cd pomodoro-timer



\# Create build directory

mkdir build

cd build



\# Configure and build

cmake ..

cmake --build .



\# Run the application

./PomodoroTimer  # Linux/macOS

\# or

Debug\\PomodoroTimer.exe  # Windows
